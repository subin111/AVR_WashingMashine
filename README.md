## 1. 개요
   
* 추진 배경 및 기대효과
  
  -ATmega128A 보드로 세탁기 구현
  -AVR 보드인 ATmega128a의 학습과정을 총정리 하는 팀프로젝트
  -프로토타입 제작을 통해 전자제품 구성에 대한 이해도를 높임
   

* 구현 목표 및 내용
  
  -여닫이 문 닫혔을 시에만 모드 시작
  -전원 버튼 구현
  -세탁기 기능 구현 [세탁, 헹굼, 탈수]
  -일시정지/ 시작 버튼 구현
  -모드 종료 시 종료 음 구현

  
* 개발 구성도

* 개발 환경 및 장비
   
  Board : ATmega128a   
  Use to : 초음파 센서, 부저, lcd, dc모터    
  Language : C    
  Program : CodeVisionAVR
